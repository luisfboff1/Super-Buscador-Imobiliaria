# Domain: facts

## Purpose
Stores stable project-level facts, environment details, and repository-wide operational knowledge.

## Scope
Included in this domain:
- Repository-wide technical choices
- Persistent operational facts about the project
- Environment and tooling facts relevant to project work

Excluded from this domain:
- Feature-specific implementation details
- Transient task plans without durable project value

## Ownership
Project repository maintainers

## Usage
Use this domain for concise facts that help future work across the repository.
