# Domain: architecture

## Purpose
Contains implementation and operational knowledge for the web scraping pipeline, crawling behavior, extraction quality, and benchmark workflows used in the project.

## Scope
Included in this domain:
- Crawler behavior and extractors
- Shadow mode and benchmark workflows
- Scraping reliability adjustments
- Operational constraints affecting scraping execution

Excluded from this domain:
- General product UI behavior
- Authentication flows unrelated to scraping

## Ownership
Scraping and ingestion implementation

## Usage
Use this domain for technical scraping pipeline behavior, reliability fixes, and benchmark evidence.
